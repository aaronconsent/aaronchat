Deuces Wild Poker Club

A private poker club in Huntsville, TX — rebuilt from a page builder into a fast, self-owned site engineered to be found and to fill seats.

Visit the live site ↗ ← All work

The brief

Deuces Wild was locked inside a page builder with organic search as its only real channel. It needed a site it owned, built to rank in a market full of bigger-city competitors.

What we built

  • Replaced a Durable page-builder site with a hand-built, conversion-first static site on Cloudflare Workers — no builder, fully owned.
  • Shipped 40+ pages including 20 local “poker near {town}” pages with real geographic hooks and schema.
  • Built the SEO/AEO layer — llms.txt, structured data, a 301 map, and explicit AI-crawler access so answer engines can cite it.
  • Stood up an autonomous B2B events-outreach engine (Resend, 3-touch sequences) and an SMS opt-in that writes straight to the owner’s spreadsheet.
